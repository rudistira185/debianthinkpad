🔐 Setup VPN L2TP/IPsec MikroTik (RouterOS v7)
📌 Deskripsi

Proyek ini menjelaskan cara mengkonfigurasi server VPN L2TP dengan IPsec pada MikroTik RouterOS v7 untuk akses jaringan secara aman.

L2TP berfungsi sebagai tunneling (jalur), sedangkan IPsec menyediakan enkripsi dan keamanan data.


🚀 Use Case
- Akses remote ke jaringan kantor/lab
- Akses server dari luar jaringan
- Tunnel jaringan internal


📊 Kelebihan & Kekurangan

✅ Kelebihan
- Mudah dikonfigurasi
- Support luas (Windows, Android, iOS)

❌ Kekurangan
-Performa di bawah WireGuard
- Kurang aman jika salah konfigurasi
