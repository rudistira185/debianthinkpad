# MikroTik PPPoE Server (RouterOS v7)
📌 Deskripsi

Repositori ini berisi konfigurasi sederhana untuk membuat PPPoE Server di MikroTik RouterOS v7.

PPPoE (Point-to-Point Protocol over Ethernet) digunakan untuk autentikasi pengguna (username & password) serta pemberian IP address secara dinamis, umum dipakai oleh ISP.


simulasi dilakukan dengan menggunakan software GNS3
