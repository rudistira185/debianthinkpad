#BGP Basic Lab (3 Router - Jakarta, Bandung, Surabaya)
Overview

Lab ini mendemonstrasikan implementasi Border Gateway Protocol (BGP) pada tiga router yang mewakili tiga lokasi berbeda:
- Jakarta
- Bandung
- Surabaya

Setiap router berada pada Autonomous System (AS) yang berbeda dan saling bertukar routing menggunakan eBGP.

Lab dijalankan menggunakan GNS3 dan MikroTik RouterOS v7.

#Objectives

- Memahami konsep dasar BGP
- Memahami eBGP (external BGP)
- Membangun BGP session antar router
- Mengiklankan network menggunakan BGP
- Memverifikasi pertukaran route antar AS
