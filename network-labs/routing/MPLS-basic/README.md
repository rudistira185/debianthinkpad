📌 Overview

Implementasi dasar MPLS menggunakan 3 router MikroTik (RouterOS v7) dengan OSPF sebagai IGP dan LDP untuk distribusi label.

🧠 Konsep

MPLS (Multiprotocol Label Switching) menggunakan label untuk forwarding paket, menggantikan proses lookup IP tradisional.

Keuntungan:

🚀 Forwarding lebih cepat
🎯 Jalur lebih deterministic
🧩 Digunakan untuk VPN (VPLS, L3VPN)
