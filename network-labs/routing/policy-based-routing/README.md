#Policy Based Routing (PBR) Lab - MikroTik RouterOS v7

Overview

Lab ini mendemonstrasikan implementasi Policy-Based Routing (PBR) pada MikroTik RouterOS v7.
Berbeda dengan routing biasa yang hanya berdasarkan destination, PBR memungkinkan router menentukan jalur berdasarkan kebijakan tertentu seperti source address.

Lab dijalankan menggunakan GNS3.

#Objectives

- Memahami konsep Policy-Based Routing (PBR)
- Menggunakan routing table terpisah
- Mengarahkan traffic berdasarkan source address
- Memverifikasi jalur routing berbeda untuk traffic berbeda
