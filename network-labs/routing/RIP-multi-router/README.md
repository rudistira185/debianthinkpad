#RIP Lab (2 Hops - 3 Router)
Overview

Lab ini mendemonstrasikan implementasi Routing Information Protocol (RIP) pada tiga router menggunakan MikroTik RouterOS v7.

RIP adalah routing protocol berbasis distance-vector yang menggunakan hop count sebagai metric utama.

Lab dijalankan menggunakan GNS3 untuk memahami konsep dasar routing sederhana.

#Objectives

- Memahami konsep dasar RIP
- Memahami hop count sebagai metric
- Mengamati pertukaran routing antar router
- Memverifikasi jalur routing berdasarkan hop
