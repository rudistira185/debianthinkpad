VLAN Trunking MikroTik v7
📌 Deskripsi

Proyek ini menjelaskan implementasi VLAN Trunking pada MikroTik RouterOS v7 menggunakan fitur bridge VLAN filtering.

VLAN Trunking memungkinkan beberapa VLAN dikirim melalui satu interface fisik menggunakan tagging IEEE 802.1Q.

🧠 Konsep Dasar
- VLAN (Virtual LAN): Segmentasi jaringan Layer 2
- Trunk Port: Port yang membawa banyak VLAN (tagged)
- Access Port: Port yang membawa 1 VLAN (untagged)
- 802.1Q Tagging: Penandaan frame dengan VLAN ID


📈 Use Case
- Inter-VLAN routing
- Segmentasi jaringan
- ISP / jaringan kampus
- Lab networking (GNS3 / EVE-NG)
